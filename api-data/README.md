# Users Data Dashboard

A simple React application that fetches user data from a public API and displays it in a responsive UI. The project demonstrates API integration, loading states, and error handling using Axios.

## Features

- Fetches data from a public API
- Displays user information in clean cards
- Handles loading state
- Handles error state
- Responsive design for mobile and desktop

## Technologies Used

- React
- Axios
- Tailwind CSS
- JavaScript

## API Used

This project uses the JSONPlaceholder public API.

https://jsonplaceholder.typicode.com/users
